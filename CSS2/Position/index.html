Relative

Zachowuje się jak static jednak mozna go przesuwać uzywając: top / right / bottom / left.

1. Wyciąga element ponad flow dokumentu, dlatego przy przesuwaniu, nie wpływa na pozycje innych elementów

2. Wyciąga element z flow

3. Najczesciej uzywa się go jako "parenta", poniewaz przez wyciagniecie z flow dokumentu trudno stylowac rzeczy dookola niego

Absolute

Pozycjonuje element, jakby został kompletnie usunięty.

1. Usuwa element z flow dokumentu, dlatego przy renderowaniu zostaje pominiety

2. Jest uzyteczny w momencie kiedy chcemy coś ustawić w danej pozycji a nie chcemy ruszać innych elementów

3. Element pozycjonuje się w stosunku do "parenta" lub całego dokumentu

Fixed

Pozycjonuje bardzo podobnie do absolute, jednak jest pare roznic.

1. Ignoruje pozycję relatywną "parenta"

2. Zawsze bazuje na całym elemencie HTML"

3. Zawsze zostaje w tym samym miejscu podczas scrollowania

Sticky

Jest kombinacją pozycji relative i fixed.

1. Nie ignoruje pozycji "parenta"

2. Zostaje w tym samym miejscu podczas scrollowania
