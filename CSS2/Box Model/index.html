Notes

Everything in CSS is a box of either a rectangular shape or a square shape

No matter is it a text, span, div, button - every single element in CSS is a box and these boxes have diffrent parts.

You have the actual content of the box which in our case is text1 and text2

We have the padding, border and the margin of the box


The easiest thing to understand is the actual content of the box if you do nothing to a box at all, The content of the box is just going to fill up the space inside of the box and the box will be no larger than the content but if u start adding different things to the box then you can start to expand the size of your box. One of the easiest ways to make your box bigger is to use padding


Padding - występuje w tle elementu i obejmuje cały obszar az do borderu. Obejmuje cały content, który jest najbardziej zagniezdzony w box model. Padding obejmuje tło, dlatego większość buttonów jest stylowanych przy jego pomocy. Pozwala to na poszerzenie buttonu, przez co jest większy niz text. Najczęściej uzywany do tworzenia tła.


Border - występuje on wokół paddingu naszego elementu. Tak więc mamy w tym wypadku 20px paddingu, wokół ktorego występuje 20px borderu


Margin - Jest to ostatni element występujący w Box Model. Po dodaniu 20px marginesu mozemy dostrzeć, ze nasz box zwiekszyl się o 20px poza granicami borderu. Marginesu zazwyczaj uzywamy do oddzielenia od siebie dwóch elementów

Jezeli mamy dwa boxy, ktore sa przy sobie, jezeli w jednym ustawimy margines np. 30 i w drugim 30 to nie beda one odsuniete od siebie o 60px tylko o 30px, poniewaz marginesy zapadaja sie jeden w drugim. Zawsze bedzie uzyty margin, ktory ma wieksza wartosc.

Height i Width są sumą, czyli jezeli mamy ustawione je na 100px, to dodajac do tego 20px paddingu i 20px borderu, otrzymujemy height i width w postaci 180px. Margin się nie wlicza w wielkosc elementu.

Jezeli chcemy to zmienic i chcemy aby height: 100px i width:100px były właściwymi wartościami, uzywamy wtedy komendy box-sizing: border-box; - która sprawia, ze border i padding zamiast się dodawać to odejmują nam rozmiar od naszego contentu, czyli gdy width=100px i height=100px a padding=20px i border=20px // height i width same w sobie wynoszą po 20px, natomiast w sumie width/height + padding + border = 100px x 100px czyli tak jak mamy ustawione w css
